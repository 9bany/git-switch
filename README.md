## How to install 
### 1. MAC OS
```
  brew tap 9bany/swgit

  brew install swgit
```
### 2. Linux

Download the latest version of `swgit` from the [releases](https://github.com/9bany/git-switch/releases) page and then run:
```
$ cp -i swgit-linux /usr/local/bin/swgit
```

### 3. Nodejs 
```
npm i @browng/swgit
#or
yarn add @browng/swgit
```
