- [Bug reports](./docs/bug_report.md)

## Features 
- Switch 2 accounts github.
- Support ssh key: create and manager ssh keys.
- Auto switch then current account not allow for the repository.
## Development 
- Install 
```
make install
```
- unInstall 
```
make uninstall
```
- Run test cases
```
make test
```
## CLI 
- Help
```
swgit --help
```
